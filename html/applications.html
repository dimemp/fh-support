Homepage ▷ Organizers ▷ Applications

•To view and evaluate the applications of your program, from the home page select founders & organizers. On the new page you are taken to, under the section called Applications select All applications. Then select the startup you want, click the three tick boxes and evaluate. Evaluate by category and finally, click the save button.

.

•To move program applicants to the finalist group, select the applicant you are interested in, click the three tick boxes and select finalists under the Move section.

•To view the users of the program, in the all applications tab click users

•On the new page you have been redirected to you can see all users

•If you want to edit and preview the custom form for the applicants, in the same page press the custom form button. Press the review form button to preview the form.

.

•In order to edit the form, there are plenty of edits that you can perform. You can press the Add field button and choose from the drop-down-menu the category you'd like to add. Then, you can fill the fields with the Field label and the Reference value field name you like. Finally, press the save button. You can also add or remove options in an existing category with the buttons with the plus and minus symbol. Last but not least, if you want to delete a category, press the delete button (bin) next to the category you chose.

.

•Start/stop accepting applications for your program by navigating to the settings tab. Enable or disable the toggle button and press the save button.

.

•In order to share your program via link, move to the settings section of the same page. Then, press the copy link button. You can invite startups to apply for All applications phase through this link.

.

•You can check on the finalists by clicking on Finalists tab from the left column.

.

•You can re-evaluate the finalists of the program by using the same steps of evaluation for all applications. The only difference is that now you select the finalists tab instead of All applications.

.

•Move a finalist back to simple application phase by clicking move all applications next the startup you desire.

.

•Check on the user finalists by clicking Users in the Finalists tab.

.

•You can add/remove/edit evaluation criteria for applicants by clicking settings in Applications tab

.

•In the new page, you can either add new criteria by typing the criteria name under the field titled Create new criteria and press the Add criteria button, or edit an existing criteria by selecting the edit button (pencil). Last but not least, delete a evaluation criteria by clicking on the delete button (bin).

.
