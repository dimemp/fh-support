Homepage ▷ Organizers ▷ Virtual Program

•To set group and mentor in the virtual program, from the left column and in the Virtual program section, select Startups

.

•Then, select the startup you want and click settings.

•On the new page you are taken to, you can set/change group or mentor or both. Finally, hit the save button to save the changes you made.

•To modify the setup of the virtual program, in the virtual program section select Setup.You can then add/remove the categories you wish to include in your virtual program.

•To create a new group category, in the left column select groups.

•In the field titled Create new group, type the name of the new group and press the + New group button.

•To add/modify the announcements of the virtual program, select Announcements from the left column.

•In the field titled Announcements, type or modify the announcement you want and finally click the save button.

•To add/modifylibrary content, select library from the left column.

•In the field titled library, type or modify the content and finally press the save button.

•To start a live session we first have to have the stream linked (see bellow for instructions), we can then give a Title to the Live session by selecting the text field and pressing save when done. We can even add a comment by selecting the filed and when ready pressing the Enter key or by clicking the send icon.

•To add a previous session, from the Live section we select Previous Lives. We can then add a Title and a link for the session, press Save when ready. Pressing the link should redirect us to the video.

•To change stream provider/link for our live sessions, from the Live section we select Settings. We then select stream provider and depending the provider, we copy+paste the special ID string in the URL of the video to the field requested. When done press Save.

•In order to add/edit/remove channels to the forum, select Forum from the left column, and choose Channels. If you want to create a new channel, in the field titled Create new channel, type the channel name you like. If you want to edit a channel that already exists, under the tittle Channels, press the edit button (pencil) next to the channel you want to edit.

•Apply the changes you like and press the save button. If you want to delete a channel, under the same section, choose the channel you want to erase and press the delete button (bin).

•Create a new post in the forum by clicking new post in the forum section. Choose the channel you want to post to, type your question and more information about it and click the post it button.

•Add a comment to a post by clicking on a post. In the new page, type your comment in the field titled New comment and press the button Add comment.

•In order to add/edit/remove an external partner, select 1-on-1 from the left column. If you want to add an external partner, choose Add new. Type the data of the partner you want to add, and finally press the Add new person button. If you want to edit the data of an existing external partner, press the View all button. Choose the partner you want to edit and press the edit button. After you finish the editing, press the save button.

•If you want to remove an external partner, in the same section choose the partner you want to remove and press the edit button. Finally, press the delete button at the top of the page and click the button save.

•In order to check progress reports of each organization, select Progress from the left column. Press the Progress reports button, choose the organization you want to check and press the All updates button. In the new page you can see the progress you want. If you want to edit and preview the Updates form for the organizations, in the same page press the Updates form button.

•Press the preview form button to preview the form. In order to edit the preview form, there are plenty of edits that you can perform. You can press the Add field button and choose from the drop-down-menu the category you'd like to add. Then, you can fill the fields with the Field label and the Reference value field name you like. Finally, press the save button.

•You can also add or remove options in an existing category with the buttons with the plus and minus symbol. Last but not least, if you want to delete a category, press the delete button (bin) next to the category you chose.

•In order to create/edit/remove perks, select Perks from the left column. Under the Add new title, you can fill in the empty fields and browse the logo of your liking. Finally, press the Create new perk button. If you want to edit an existing perk, select the View all option. In the new page, choose the perk you want to edit and press the edit button.

•Apply your desired changes and press the Save button. Last but not least, in the same edit page, press the delete button to remove the perk you chose.

•In order to send email to all participants, select Email from the left column. In the new page you were transferred to, fill the Email subject and the body of the email, and press the send button.

•You can check on your mentors and evaluators by clicking View all from the left column. In the new page you can overview the mentors and the evaluators. In order to invite evaluators/mentors, copy the link under the title Invite evaluators/mentors from the left column. You can share it to any evaluator or mentor of your liking.
